# Medical Imaging Classification CNN
Implements CNN using TensorFlow/Keras to classify medical images by body part and imaging type

# Dataset: Medical MNIST
  - Classes: Abdomen CT, Breast MRI, Chest CT, Chest X-ray (CXR), Hand X-ray, Head CT
  - Source: https://www.kaggle.com/datasets/andrewmvd/medical-mnist
  - dataset not included in repo. When running in Colab, place dataset in '/content/archive'.

# Files
  - `Medical_Classification.ipynb`: Full training, evaluation, and visualization notebook
